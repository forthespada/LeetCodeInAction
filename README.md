**3、学习思考**

力扣上的题目越来越多，全部刷完基本不现实，所以我在刷题过程中并没有注重**数量**，而更多的注重题的**质量**。

最后选择按照下面这**14**个类别刷题，一共刷了将近**600+**，后来二刷、三刷过后又筛掉了一些，现在只剩下这些比较经典的题。

刷题顺序是按照标签刷，每个标签下又分为easy、medium、hard三个等级~
